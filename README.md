# 📄 PDF Viewer UI

Um projeto moderno de **Visualizador de Documentos PDF**, desenvolvido para oferecer uma experiência simples, organizada e intuitiva na visualização de arquivos digitais diretamente no navegador.

Ideal para sistemas acadêmicos, plataformas corporativas, gerenciadores de documentos e aplicações web que necessitam exibir arquivos PDF de forma profissional.

---

## 📸 Preview do Projeto

Interface contendo:

- Visualização integrada de documentos PDF
- Barra de ferramentas completa
- Controle de zoom
- Navegação entre páginas
- Opções de download e impressão
- Área de leitura responsiva

---

## ✨ Tecnologias Utilizadas

- **HTML5**
- **CSS3**
- **JavaScript**
- **PDF.js**

---

## 🎨 Características do Design

- Layout limpo e profissional
- Interface inspirada em leitores de PDF modernos
- Navegação intuitiva
- Estrutura organizada e responsiva
- Controles acessíveis e funcionais
- Experiência otimizada para leitura

---

## 🚀 Objetivo do Projeto

Praticar o desenvolvimento de interfaces para gerenciamento e visualização de documentos digitais, explorando integração com bibliotecas de leitura PDF e construção de aplicações web funcionais.

---

## 📚 Funcionalidades

- Exibição de documentos PDF no navegador
- Controle de zoom (ampliar e reduzir)
- Navegação entre páginas
- Download do documento
- Impressão direta
- Barra de ferramentas interativa
- Rolagem horizontal e vertical
- Compatibilidade com diferentes tamanhos de tela
